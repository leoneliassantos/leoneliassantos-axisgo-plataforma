Enviar o Core atualizado para o GitHub

Copie o comando e cole no terminal do VS Code. Isso atualiza o Batux e deixa o Core pronto para a MC.

Passo a passo:

  1. No VS Code: menu Terminal → New Terminal.
  2. Clique em Copiar comando acima.
  3. Clique na faixa preta do terminal e cole (Ctrl+V ou botão direito → Paste). Aperte Enter.
  4. Deve aparecer no final algo como main -> main. 🎉
