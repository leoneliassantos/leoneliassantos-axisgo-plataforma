Passo 2B — Tornar seu usuário administrador

Só faça isto depois de já ter criado seu usuário em Authentication (parte 2A).

Este comando promove o único usuário que existe a administrador — perfeito para o seu primeiro acesso.

O que fazer com o código:

  1. Volte ao SQL Editor (a mesma tela verde de antes).
  2. Apague o que estiver escrito, clique na área branca e cole com Ctrl+V.
  3. Clique em Run (ou Ctrl+Enter).
  4. Deve aparecer “Success” — pronto, você virou administrador! 🎉
