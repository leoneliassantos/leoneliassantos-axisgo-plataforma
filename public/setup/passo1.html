Passo 1 — Copiar o código do banco

Clique no botão abaixo. Ele copia todo o código automaticamente para você colar no Supabase.

Se o botão não funcionar, clique dentro do quadro abaixo, aperte Ctrl+A (seleciona tudo) e depois Ctrl+C (copia):

Depois de copiar, faça no Supabase:

  1. Volte para a aba do SQL Editor.
  2. Clique na área grande em branco e cole com Ctrl+V.
  3. Clique no botão verde Run (canto inferior direito) ou aperte Ctrl+Enter.
  4. Deve aparecer “Success. No rows returned” — deu certo! 🎉
