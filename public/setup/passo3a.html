Nome exato desta função: admin-create-user

Como criar no Supabase:

  1. No menu esquerdo, clique em Edge Functions.
  2. Clique em Deploy a new function → escolha a opção Via Editor (criar pelo navegador).
  3. No campo de nome, digite exatamente: admin-create-user
  4. Apague o código de exemplo que vier, clique no editor e cole com Ctrl+V (use o botão acima pra copiar).
  5. Clique em Deploy function (ou Deploy). Espere aparecer que foi publicada. 🎉
