Enviar o código da plataforma para o GitHub

Você vai colar um comando no terminal do VS Code. Ele usa o seu login do GitHub (que já está ativo no VS Code).

Passo a passo:

  1. No VS Code, abra o terminal: menu Terminal → New Terminal.
  2. Clique no botão “Copiar comando” acima.
  3. Clique dentro do terminal (a faixa preta) e cole com Ctrl+V (ou botão direito → Paste). Aperte Enter.
  4. Se aparecer uma janelinha do VS Code pedindo para entrar no GitHub / autorizar, clique em Allow / Sign in e confirme (você já está logado como leoneliassantos).
  5. Aguarde. Quando terminar, deve aparecer algo como “branch 'main' set up to track 'origin/main'” e várias linhas de “done”. 🎉
